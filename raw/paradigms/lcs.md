# Longest Common Subsequence (LCS)

## 부제목
두 수열의 가장 긴 공통 부분수열 구하기

## 한 줄 정의
LCS(최장 공통 부분수열)는 두 수열 X와 Y에서 공통으로 나타나는 부분수열 중 가장 긴 것을 찾는 문제이다. (연속적이지 않아도 순서만 유지되면 인정)

## DP 점화식 매핑
```text
c[i][j] = 0                                    (i=0 또는 j=0)
c[i][j] = c[i-1][j-1] + 1                     (xᵢ = yⱼ)
c[i][j] = max(c[i-1][j], c[i][j-1])           (xᵢ ≠ yⱼ)

```

## 복잡도 임계치

* 시간 공간 복잡도 모두 `O(mn)` 구조 형성 (m과 n은 각각 매칭할 두 수열의 길이 벡터)
