# Algorithm Analysis

## 부제목
알고리즘의 정확성, 시간, 공간, 최적성을 평가하는 방법론

## 한 줄 정의
알고리즘 분석이란 주어진 알고리즘의 정확성(correctness), 시간 복잡도(time complexity), 공간 복잡도(space complexity), 최적성(optimality)을 체계적으로 평가하는 과정이다.

## 분석 단계
- **기본 연산 선정** — 알고리즘의 핵심 반복 연산 식별
- **복잡도 함수 수립** — 기본 연산 횟수를 n의 함수로 표현
- **점근 분류** — Big-O, Ω, Θ 표기로 분류

## Problem Solving 단계 (CH01)
```text
1. Problem  →  2. Strategy  →  3. Algorithm
4. Analysis (Correctness, Time & Space, Optimality)
5. Implementation  →  6. Verification
```
